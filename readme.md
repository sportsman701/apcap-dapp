1. Install packages by `yarn` or `npm install` in the root directory.

2. Set config vars

Put variables like vote period, choice1, choice2 in `.env` like below.
REACT_APP_PERIOD = 15
REACT_APP_CHOICE1 = Yes
REACT_APP_CHOICE2 =  No

3. Run Ganache by `npm run ganache` or `yarn ganache`

4. Run dapp by 'npm run start` or `yarn start`
It will compile the smart contract automatically and run dapp based on its ABI.

5. Enjoy the app!
